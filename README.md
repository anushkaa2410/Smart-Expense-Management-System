# Smart-Expense-Management-System
Smart Monthly Expense Decision System – A C++ finance management application that uses Greedy Algorithms, Dynamic Programming, Backtracking, Divide &amp; Conquer, Merge Sort, and Binary Search to analyze expenses, optimize savings, and provide intelligent financial recommendations.

## 📌 Project Overview

The **Smart Monthly Expense Decision System** is a C++ based financial management application developed as part of the **Design and Analysis of Algorithms (DAA) Lab Project**.

The system helps users manage their monthly expenses, analyze spending behavior, generate optimized savings plans, predict future expenses, and make informed financial decisions using multiple algorithmic approaches.

Unlike traditional expense trackers, this project applies advanced DAA concepts such as **Greedy Algorithms, Dynamic Programming, Backtracking, Divide & Conquer, Merge Sort, and Binary Search** to provide intelligent financial recommendations.

---

## Objectives

- Record and manage personal expenses.
- Categorize expenses into essential and non-essential spending.
- Analyze monthly spending patterns.
- Generate optimized savings plans.
- Predict future financial conditions.
- Provide smart expense reduction suggestions.
- Demonstrate real-world applications of DAA concepts.

## Algorithms Implemented

### 1. Merge Sort
Used for sorting expense records efficiently.

**Time Complexity:** O(n log n)

---

### 2. Binary Search
Used for fast searching of expense records after sorting.

**Time Complexity:** O(log n)

---

### 3. Dynamic Programming (0/1 Knapsack)
Used in the Savings Planner to generate optimal expense reduction strategies.

**Time Complexity:** O(n × W)

---

### 4. Backtracking
Generates multiple valid expense-cut combinations to achieve savings goals.

---

### 5. Divide & Conquer
Used for spending trend analysis and expense aggregation.

---

### 6. Greedy Algorithm
Prioritizes expenses and identifies areas where spending can be reduced.
